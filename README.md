# Strapi application

A quick description of your strapi application

This is a strapi simple project using mongo database and APIs generated by Swagger.
It creates API for Organizations and the users in the organizations to be managed.  The management function involves CRUD operations for the two entities and also defining organization roles for the users. Organizations can also post job and there is a custom functionality to find the first organization in the list of organizations managed by the application in the findFirst API route.
The project is hosted live on heroku here: https://strapi-organization-manager.herokuapp.com/
The admin page is: https://strapi-organization-manager.herokuapp.com/admin
